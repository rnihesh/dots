# Scripts Collection

A collection of useful bash scripts for productivity and automation. This repository contains executable scripts and configuration files organized into folders.

## Scripts (scripts/)

| Script    | Description                                                                      |
| --------- | -------------------------------------------------------------------------------- |
| ollaman   | Interact with Ollama AI models via terminal                                       |
| openf     | Fuzzy open files/folders with default app                                         |
| codef     | Fuzzy open files/folders in VS Code                                               |
| cleanx    | Find and delete build/dependency junk dirs with size preview and confirm prompt   |

## Configuration Files (conf/)

| File      | Description                                           |
| --------- | ----------------------------------------------------- |
| tmux.conf | Tmux configuration with Catppuccin theme and keybinds |
